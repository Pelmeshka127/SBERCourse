# SBERCourse

Для работы проекта потребуется [flex](https://gnuwin32.sourceforge.net/packages/flex.htm) и CMake версии не меньше 3.8.

## Задание 1: лексический анализ

Установка проекта:

```
git clone git@github.com:Pelmeshka127/SBERCourse.git

cd SBERCourse
```

Запуск:

```
cmake -B build

cd build

make

./stasyan 
```
